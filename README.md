# Inequality, Economic Development and Interpersonal Trust (ESS 2023)

This project analyzes the relationship between income inequality, economic development, and interpersonal trust in Europe using ESS Round 11 (2023) and Eurostat data.

## Structure

- notebooks/01_data_prep.ipynb → data cleaning and merging
- notebooks/02_analysis.ipynb → regression analysis and figures
- data/processed/ → cleaned merged dataset
- results/ → regression tables and plots
- paper/ → final research paper

Note: ESS raw data is not included due to licensing restrictions.
